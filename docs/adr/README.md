# Architecture Decision Records (ADRs)

This project uses ADRs in the Michael Nygard format.

## Naming
- Use sequential IDs: `0001`, `0002`, ...
- Use kebab-case slugs: `0001-use-java-spring-boot-thymeleaf-maven.md`

## Status values
- Proposed
- Accepted
- Superseded by ADR-XXXX
- Deprecated

## Template
Use the template in `docs/adr/_template.md`.
